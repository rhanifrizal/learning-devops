# Company Server

## Overview

A simulated Linux server environment used to practice common DevOps and Linux administration tasks.


## Features

- Server monitoring
- Health checks
- Service inspection
- Log cleanup
- Backup scripts
- Deployment simulation


## Technologies

- Ubuntu
- Bash
- Linux
- systemd


## Project Structure

```text
company-server/
├── README.md
└── web-app/
    ├── Dockerfile
    ├── app.py
    ├── config.yml
    ├── logs/
    └── scripts/
        ├── monitor-server.sh
        ├── cleanup-logs.sh
        └── deploy.sh
```


## Scripts

| Script              | Purpose                                      |
| ------------------- | -------------------------------------------- |
| `monitor-server.sh` | Displays server health information           |
| `cleanup-logs.sh`   | Creates a compressed archive of old logs     |
| `deploy.sh`         | Simulates an application deployment workflow |
